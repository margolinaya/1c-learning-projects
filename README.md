# 1c-learning-projects

# Учебные проекты по 1С:Предприятие 8.3

Репозиторий с моими учебными проектами по разработке на платформе 1С:Предприятие 8.5.

---

## Проекты

### [Тайм-Трекер](./timetracker/)
Учёт времени по задачам и проектам.
- **Документы:** Регистрация времени
- **Регистры:** Учёт времени по задачам
- **Отчёты:** Сводка по задачам, детализация по дням
- **Обработка:** Таймер для засечения времени

---

### [Учёт бюджета](./budget/)
Учёт личных финансов с план-факт анализом.
- **Справочники:** Статьи бюджета, Счета
- **Документы:** Операции, План бюджета
- **Регистры:** Остатки денег, факт и план бюджета
- **Отчёты:** Остатки по счетам, план-факт

---

## Используемые механизмы платформы 1С

- Справочники, документы, регистры накопления и сведений
- Запросы, в том числе с соединениями и временными таблицами
- Система компоновки данных (СКД)
- Управляемые формы, обработчики на клиенте и сервере

---

## Цель проекта

Закрепление практических навыков разработки на 1С и создание портфолио для дальнейшего трудоустройства.

---

## Статус

Проекты учебные, находятся в процессе изучения и доработки.
